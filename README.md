# Execute e implante seu aplicativo AI Studio
Isso contém tudo o que você precisa para executar seu aplicativo localmente.

Visualize seu aplicativo no AI Studio: https://ai.studio/apps/drive/1WU3kw0iQeo51VIhJx63MlXaYLb7ywMrW

## Executar Localmente

**Pré-requisitos:**  Node.js


1. Instalar dependências:
   `npm install`
2. Defina a `GEMINI_API_KEY` in [.env.local](.env.local) para sua chave API Gemini
3. Executar o aplicativo:
   `npm run dev`
